# FastMqtt
<img src="img/fastmqtt.jpg" style="margin-bottom:8px;">
<p>
FastMqtt是一款开源的MQTT Web客户端，用于快速测试MQTT功能，双击浏览器打开，无需任何依赖，可以集成到Java等项目中去。
